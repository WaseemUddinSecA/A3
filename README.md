Interactive Test:-
This project uses HTML, CSS, and JavaScript to create a straightforward interactive quiz. Football-related multiple-choice questions are included in the quiz, along with instant feedback on the answers you choose. The user is presented with their final result after finishing the quiz.
Features:- 
• Dynamic Question Display: Each question is shown one at a time after being retrieved from an array. 
• Instant Feedback: Users are informed right away if their response was accurate or not. 
• Score Calculation: The quantity of right answers determines the score. 
• Quiz Completion: The user sees their final score after answering every question. 
Technologies Used:-
• HTML: Offers the quiz's content and structure. 
• CSS: Styles the colors, design, and simple animations of the quiz interface. 
• JavaScript: Uses quiz logic to display questions, validate answers, and count points.
